# AI Usage Disclosure

This project used AI assistance in the following SDLC activities:

1. Requirement-to-backlog translation for the user stories document.
2. Initial REST API skeleton generation (controllers, services, DTOs, entity modeling).
3. Security configuration drafting (JWT filter and role-based access rules).
4. Unit test generation for password policy and order flow business rules.
5. Documentation drafting (`README.md`, implementation notes).

## Human Review and Validation
- All generated code and tests were reviewed by the developer.
- Security rules and endpoint behavior were manually verified.
- Build/test execution was used to confirm runnable status.
